# Task 10.5 — Multi-ticker display

**Depends on:** Tasks 07–10

## Steps (in order)

1. **Backend batch endpoint** — `GET /stocks/prices?tickers=AAPL,MSFT&...` with asyncio.gather
2. **API client** — add `getMultiStockPrices(tickers[], opts)`
3. **useStocks refactor** — `selectedTickers[]`, `pendingTickers[]`, `priceData: { [ticker]: [] }`, `applyTickers()`
4. **TickerSelector** — color tags, X to remove, searchable dropdown (max 5), Apply + Clear all
5. **StockChart** — one Line per ticker, % change Y-axis, clickable Legend, multi-ticker tooltip
6. **StockTable** — Ticker column with color dot, flatten rows, per-ticker filter checkboxes
7. **NotesSidebar** — `useNotes(tickers[])`, grouped by ticker, collapsible sections, ticker dropdown in form
8. **App.jsx + CSS** — wire all updated props, styles for tags/dots/collapsible

## Color palette
```js
const TICKER_COLORS = ["#6366f1", "#f59e0b", "#10b981", "#ef4444", "#3b82f6"]
```
Same color used in chart lines, tags, table dots, note section headers.

## Done when
- Selecting 3 tickers shows 3 lines on chart (% change normalized)
- Table shows rows from all 3 tickers, filterable
- Notes sidebar groups by ticker with collapsible sections
