Here is the prompt:

You are an aspiring financial wizard wanting to study data about S&P 500 companies. You have decided to build a full stack web app to do so, and have a 4-step master plan of requirements:


    You first need to acquire S&P 500 stock price data. You need to keep this data as fresh as possible given the S&P updates itself in real-time.


    To make sense of this data, you want to be able to represent it in tables and charts. You should have some data manipulation capabilities built in (eg: zooming into chart sections or filtering / sorting on key columns et cetera).


    You also want to be able to take notes regarding the data that is tagged to certain dates / companies. For example, if you notice an unusual spike or decline in stock price for a specific date, you want to be able to jot down thoughts about what may have happened.


    [Recommended Bonus] You want to leverage AI to help you build charts/tables and/or interact with charts/tables (eg: “Which companies had the steepest decline in stock price when COVID struck?”, or even things like “Could you create a table for the top 10 companies from 2020 to 2021 in terms of market cap?”). 


You have full autonomy over what data you depict / how you architect this solution. This is by design because instead of being prescriptive, we want to see how you think through problems, make decisions and balance trade-offs.


Submission instructions:

You have 4 hours to submit this assignment. Please share with us your code, instructions on how to run it, and any additional materials via a Zip file or Github link (if using Github, ensure Tetrix is not referenced in the repository name, README file or elsewhere). Also, please answer these questions (brief responses only - this should not take you more than 10 minutes): 

    Describe your process for completing the project. Include details on assumptions made and key decisions taken / tradeoffs made for each step (if relevant).

    Are you happy with your solution? Why or why not?

    What would you do differently if you got to do this over again or had more time?
